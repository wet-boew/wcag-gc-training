--- title: SC 2.4.13 Page Break Navigation ---

.pg-brk-aft

The purpose of this class is to force a page break on print view right after the element on which it's applied. In order to successfully encounter WCAG 2.2 success creteria on page break navigation (SC-2.4.13), users must have a way to locate page breaks via page break locators.

Page break locators

When applying this class, the next element on the page must have a unique id and a way to navigate to it such as an anchor link must be abailable on the page.
